# Language Translator 🌐

> A real-time language translation web application with speech-to-text, text-to-speech, and document support.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## 🌐 Live Demo
[View Live Website →](https://preeti529.github.io/language-translator/)

## ✨ Features

- 🌍 Multi-language translation support
- 🎤 Speech-to-text input (voice recognition)
- 🔊 Text-to-speech output
- 📋 Copy to clipboard
- 📄 Document upload & download
- 🌓 Dark / Light mode toggle
- ⚡ Real-time translation output

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Web Speech API (speech recognition + synthesis)
- Translation REST API integration
- Ionicons for UI icons

## 🚀 Run Locally

```bash
git clone https://github.com/Preeti529/language-translator.git
cd language-translator
open index.html
```
